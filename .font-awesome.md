## Defaults

- **Style:** solid
- **Family:** classic

## Import Pattern

Icons are used as inline HTML with a `<script>` tag in each page's `<head>`:

```html
<script src="https://kit.fontawesome.com/56a48e0c30.js" crossorigin="anonymous"></script>
```

Then icons are added with:

```html
<i class="fa-solid fa-check"></i>
```

## Conventions

- Static HTML project — 8 pages, all at the project root, no build tool
- The Kit script tag is placed in each page's `<head>` immediately after the viewport `<meta>` tag
- All CSS and JS is inline in each HTML file
