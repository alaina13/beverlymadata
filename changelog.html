Changelog

What's changed on Open Beverly.

July 8, 2026
Readability rewrite and accessibility fixes

Rewrote explanatory text for easier reading. No figures, quotes, or sourcing changed.

  • History page ("How We Got Here"): simplified the structural deficit explainer, the year-by-year narrative, and chart captions.
  • Key Concepts page: simplified glossary definitions for structural deficit, General Fund, Enterprise Fund, free cash, stabilization fund, ACFR, FFC, levy limit, new growth, debt exclusion, and excess levy capacity.

Accessibility fixes:

  • Added the screen-reader chart data table pattern (used on every other chart page) to the Override Calculator's two charts.
  • Added an aria-live region to the property map's parcel detail panel so screen readers announce updates when a parcel is selected.
  • Added site-wide support for prefers-reduced-motion, disabling non-functional animations and transitions for users who request reduced motion at the OS level.
July 7, 2026
Neutrality revisions, new Capital Plan page, and map fixes

Revised the following to be more neutral:

  • Home page
  • Capital Plan page
  • History page
  • What's Next page
    • City Hall renovation section

New features:

  • Added a Feedback link to the footer on every page.
  • Added a "Request a feature" option on the About page.
  • Added the Capital Plan page.

Fixes:

  • Added a close button to the mobile map sidebar.
  • Fixed map tooltips getting cut off near the edge of the screen.
  • Fixed a bug where closing a parcel panel on the map could break subsequent searches.
  • Fixed footnote numbering on the History and What's Next pages.
  • Improved error handling when property data fails to load on the Override Calculator.
